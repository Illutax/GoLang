# Tutorial from https://gobyexample.com/hello-world

Run the programm with:
```shell
$ cd "~/git/GoLang/hello world"
$ go run hellp-world.go
Hello World!
```

To compile the code run:
```shell
$ go build hello-world.go
$ls 
hello-world.exe*    hello-world.go
```

or with
```shell
$ ./hello-world
Hello World!
```
